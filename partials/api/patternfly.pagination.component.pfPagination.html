pfPagination
directive in module patternfly.pagination

Description

Component for pagination controls used in various views (list, card, table)

Usage

as element:
<pf-pagination
       page-number="{number}"
       num-total-items="{number}"
       page-size-increments="{Array<Number>}"
       page-size="{number}">
</pf-pagination>

Parameters

ParamTypeDetails
pageNumbernumber

The current page number

numTotalItemsnumber

The total number of items in the data set. When a filter is applied, update the numTotalItems accordingly.

pageSizeIncrementsArray<Number>

(optional) Page size increments for the 'per page' dropdown. If not specified, the default values are: [5, 10, 20, 40, 80, 100]

pageSizenumber

(optional) The initial page size to use. If not specified, the default will be the first increment in the pageSizeIncrements array.
